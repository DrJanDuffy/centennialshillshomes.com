# Images Directory

This directory contains all image assets for the website.

## Required Images

Please add the following image files to this directory:

### Property Images
- `modern-home-exterior.jpg` - Modern home exterior
- `clubhouse-exterior.jpg` - Clubhouse/community building

### Community Amenities
- `community-lounge.jpg` - Community lounge area

## Image Specifications

- **Format**: JPEG or WebP (WebP preferred for better compression)
- **Dimensions**: 
  - Property images: 1200x800px minimum
  - Amenity images: 1600x1000px minimum
  - Logo: 400px width minimum (for retina displays)
- **Optimization**: Compress images for web (aim for <200KB per image)
- **Naming**: Use kebab-case (lowercase with hyphens)
