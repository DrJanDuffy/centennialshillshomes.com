## Image Specifications

- **Format**: JPEG or WebP (WebP preferred for better compression)
- **Dimensions**: 
  - Property images: 1200x800px minimum
  - Amenity images: 1600x1000px minimum
- **Optimization**: Compress images for web (aim for <200KB per image)
- **Naming**: Use kebab-case (lowercase with hyphens)
- **Alt text**: Descriptive, include location and key features
